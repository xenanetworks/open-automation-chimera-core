==============================================
Xena OpenAutomation Chimera Core Documentation
==============================================

:term:`XOA Chimera Core` provides a set of high-level APIs to administer Xena's network impairment emulator, `Chimera <https://xenanetworks.com/chimera/>`_ for test automation.

The target audience of this document is test specialists who develop and run automated test scripts/programs using Xena's `Chimera <https://xenanetworks.com/chimera/>`_. Users of this document should have the following knowledge and experience:

* Ability to program with Python language.
* Familiarity with the operating system of your development environment.
* Familiarity with Xena's `Chimera <https://xenanetworks.com/chimera/>`_.
* Working knowledge of data communications theory and practice.

.. important::

   For a brief product overview, go to `Chimera Fact Sheet <https://xenanetworks.com/wp-content/uploads/documentation/factsheet/Chimera-Factsheet.pdf>`_. 

   For complete details of how to use Chimera, go to `Chimera User Manual <https://xenanetworks.com/wp-content/uploads/documentation/tech-doc/ChimeraUserManual.pdf>`_. 

   For detailed technical specifications, go to `Chimera Data Sheet <https://xenanetworks.com/wp-content/uploads/documentation/datasheets/Chimera-datasheet.pdf>`_.
